# Architecture Hexagonale et Frontend

Pourquoi ne parle-t-on pas plus d'architecture hexagonale dans le monde React ?

## Le problème du couplage

Souvent, nos composants sont trop liés à la logique métier (ou pire, à l'API).

## La solution

Isoler le domaine. Dans ce post, je montre un exemple concret de séparation des préoccupations dans une application Next.js récente.
