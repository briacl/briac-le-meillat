# Le Bit (Binary Digit)

Le **bit** est l'unité élémentaire d'information en informatique. Il représente le plus petit fragment de donnée qu'une machine peut manipuler, correspondant aux deux états physiques possibles d'un transistor :
- **1** : Le courant passe (ON).
- **0** : Le courant ne passe pas (OFF).

### Puissance de calcul
Avec $n$ bits, il est possible de créer $2^n$ séquences différentes :
- **1 bit** = 2 possibilités (0, 1).
- **8 bits (1 octet)** = $2^8 = 256$ possibilités (de 0 à 255).

L'adressage réseau repose entièrement sur cette unité, comme on le voit dans les formats [[IPv4_vs_IPv6]]. Pour manipuler ces valeurs, on utilise la [[Conversion_Binaire_Decimal]].
