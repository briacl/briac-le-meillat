# Protocoles de Communication

La suite TCP/IP organise la communication réseau en couches, utilisant l'adressage binaire pour le transport des données.

### Principaux Protocoles :
- **Ethernet (Couche 2)** : Utilise les adresses MAC pour le transport local.
- **IP (Couche 3)** : Assure le routage global via l'adressage [[BIT_Def]] (voir [[IPv4_vs_IPv6]]).
- **TCP/UDP (Couche 4)** : Gèrent le transport fiable (TCP) ou rapide (UDP).
- **DNS** : Traduit les noms de domaine en adresses IP compréhensibles par les machines.
- **DHCP** : Distribue automatiquement les configurations IP, masques et passerelles.

L'efficacité de ces protocoles repose sur la rapidité de traitement du binaire par les routeurs et serveurs.
