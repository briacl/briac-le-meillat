# Passerelle Linux

Une **Passerelle Linux** (ou gateway) est une machine configurée pour relier deux réseaux distincts, généralement un réseau local (LAN) et un réseau extérieur (WAN/Internet). Elle assure le routage des paquets entre ces interfaces.

Pour fonctionner correctement, une passerelle Linux nécessite l'activation de l'[[IP_Forwarding]] et souvent la mise en place d'un mécanisme de [[NAT_Masquerade]] pour permettre aux clients du LAN d'accéder à l'extérieur.
