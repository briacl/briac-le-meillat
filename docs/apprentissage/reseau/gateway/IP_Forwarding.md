# IP Forwarding

L'**IP Forwarding** est une fonctionnalité du noyau Linux qui permet à la machine de transmettre des paquets IP d'une interface réseau à une autre. Sans cette option, la machine se contente de traiter les paquets qui lui sont destinés personnellement.

Cette fonctionnalité est indispensable pour transformer un serveur Linux en routeur ou en [[Passerelle_Linux]].
