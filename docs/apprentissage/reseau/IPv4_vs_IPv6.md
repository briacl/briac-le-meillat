# IPv4 vs IPv6

L'adressage IP applique les concepts de [[BIT_Def]] pour identifier les machines sur un réseau.

### IPv4 (Internet Protocol version 4)
- **Format** : 32 bits (4 octets).
- **Notation** : Décimal pointé (ex : `172.31.25.9`).
- **Capacité** : $2^{32} \approx 4,2$ milliards d'adresses.
- **Problème** : Pénurie d'adresses face à l'augmentation des objets connectés.

### IPv6 (Internet Protocol version 6)
- **Format** : 128 bits.
- **Notation** : [[Hexadecimal_Base16]] (ex : `2001:0db8:85a3...`).
- **Capacité** : $2^{128}$ adresses (environ $340$ sextillions).

Le passage de l'un à l'autre est essentiel pour l'évolution des [[Protocoles_Communication]].
